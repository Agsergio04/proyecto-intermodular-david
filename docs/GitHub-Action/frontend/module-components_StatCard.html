(static, constant) StatCard

Reusable statistic card component. Represents a single metric with title, value, icon and background color. Typically used in a grid to display multiple metrics. Features: - Supports custom color classes via `color` prop - Flexible `icon` prop (any React element or SVG) - Supports dark/light theme variations - Responsive and accessible
